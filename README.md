# hana-piece-server
## Commit Message Convention

|    Type     | Description  |
|:-----------:|---|
|   `feat`    | 새로운 기능 추가 |
|    `fix`    | 버그 수정 |
|    `ci`     | CI관련 설정 수정 |
|   `docs`    | 문서 (문서 추가, 수정, 삭제) |
|   `style`   | 스타일 (코드 형식, 세미콜론 추가: 비즈니스 로직에 변경 없는 경우) |
| `refactor`  | 코드 리팩토링 |
|   `test`    | 테스트 (테스트 코드 추가, 수정, 삭제: 비즈니스 로직에 변경 없는 경우) |
|   `chore`   | 기타 변경사항 (빌드 스크립트 수정 등) |
